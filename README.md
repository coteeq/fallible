# Fallible

_All happy requests are alike; each unhappy request is unhappy in its own way_.

## Content

- Containers
  - `Error`
  - `Result<T>` = `Error` | `T`
- Constructors
  - `Error`: `Err`
  - `Result`: `Ok`, `Fail`, `PropagateError`, etc
